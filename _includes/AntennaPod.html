Uploader votre sauvegarde AntennaPod

  1. Créer une sauvegarde dans AntennaPod
    • Ouvrez AntennaPod et touchez l’icône ☰ (menu hamburger) en haut à gauche.
    • Allez dans Préférences.
    • Sélectionnez Importation / Exportation.
    • Touchez Exporter la base de données pour générer votre fichier de sauvegarde (AntennaPod.db).
    • Choisissez le dossier de destination sur votre appareil (stockage interne ou carte SD).
  2. Importer votre sauvegarde dans la web app

    Veuillez sélectionner le fichier de sauvegarde AntennaPod (AntennaPod.db) que vous venez d’exporter.

    Vous le trouverez dans le dossier que vous avez choisi à l’étape précédente.

Remarque : Les couvertures de vos podcasts seront affichées si elles sont accessibles en ligne. En cas d’échec de chargement, une illustration générique de remplacement sera utilisée.
